# OpCore Branding Guide v2.0

## The "Lagos Blue" Aesthetic
OpCore is built on a design language we call **Lagos Blue**. It reflects the energy, professionalism, and rapid pace of the Nigerian business ecosystem.

### Core Color Palette
- **Lagos Blue (Primary)**: `#0052CC` (Vibrant, Trustworthy)
- **Deep Slate**: `#0F172A` (Professional backgrounds)
- **Success Green**: `#10B981` (NTA Compliance & Profit)
- **Warning Amber**: `#F59E0B` (Audit traps & missing evidence)
- **Error Crimson**: `#EF4444` (Risky transactions)

### Typography
- **Headings**: Inter / Outfit (Bold, assertive)
- **Body**: Inter (Clean, highly readable)
- **Monospace**: JetBrains Mono (For serial IDs and currency values)

### Component Principles
1. **Glassmorphism**: Use subtle backdrops (`backdrop-blur-md`) with `white/80` or `slate-900/80`.
2. **Rounded Corners**: Use `rounded-2xl` (1rem) for main cards and `rounded-lg` for inputs.
3. **Micro-animations**: Use `framer-motion` or standard CSS transitions for all hover states and modal entries.
4. **Information Hierarchy**:
   - Primary data points (Amounts, Status) must be bold and color-coded.
   - Secondary metadata (Dates, Categories) should use `text-muted-foreground`.

## Visual Elements
- **Logos**: Always use the SVG logo with `h-8` or `h-10`.
- **Icons**: Use `lucide-react` with a consistent stroke width of `2`.
- **Badges**: Pill-shaped with low-opacity backgrounds (e.g., `bg-blue-500/10 text-blue-600`).
